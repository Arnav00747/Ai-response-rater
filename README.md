# 🚀 AI Response Rater (RLHF Simulator)

A full-stack web application that simulates **Reinforcement Learning from Human Feedback (RLHF)**.  
Users compare two AI-generated responses, select the better one, rate it, and provide feedback — mimicking how modern AI systems are improved.

## 🛠️ Tech Stack
- Frontend: HTML5, CSS3 (Dark UI), Vanilla JavaScript
- Backend: Node.js, Express.js
- Storage: Browser LocalStorage

---

## ✨ Features
- 🔹 Prompt-based response generation  
- 🔹 Side-by-side comparison (Response A vs B)  
- 🔹 User selection + star rating system  
- 🔹 Feedback input (text-based reasoning)  
- 🔹 Data persistence using LocalStorage  
- 🔹 Export dataset as JSON (for ML training simulation)
